Freeplay Emulation 

Freeplay Emulation allows the student to interact freely with an emulation and observe the results. Because of the way this Smart Graphic has been constructed freeplay is not only possible, but completely separate from the panel element. By modifying a few very simple calls within the code it is easy to change the graphics from interactive to static and back again.

The Lights panel is located in the left forward part of the overhead panel as highlighted in the overhead diagram. In this Freeplay Emulation you can activate any of the switches and see the result on the aircraft on the right side of the screen.
