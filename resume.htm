Update on 2011-05-29: First update
More tests added (originally 102, now 145) and I did compatibility/fakeSMILe changes to many demos to improve MSIE 9 / Firefox 3.6 compatibility. Original maximum number of test points was 510, now is 725 and Firefox and Opera are still by far the most SVG/SMIL capable browsers around...
Update on 2011-02-28: Initial release
Following test is my another attempt to provide browser support evaluation based on all (well, majority, not all) files hosted on my site. You can expect more changes to come in the future.
Some demos utilize FakeSMILe to add at least some SMIL support to MSIE9 and is a way of cheating, but I feel it is good to use workarounds where there are available and easy to use.
Some of the tests are not fair to other browsers (like lmc-org or xhtml test for Batik) but I hope to solve these handicaps in the future as well...
725 points = maximum
